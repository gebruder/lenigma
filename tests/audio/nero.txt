# Three identical repetitions of the strong-signal portion of a smart-
# beep transmission (START + 0 + 1 + sustained-0 + 1 + 2, separated by
# ~1.65 s). Tones land cleanly in the FSK bands (812 / 859 / 906 /
# 2058 Hz = 0, 1, 2, START) but the per-cycle sequence isn't a complete
# 18- or 22-nibble framed message — the remaining tones in each
# transmission are below the recording's detection floor. Useful
# regression sample for confirming the decoder still finds every
# audible tone in a clean recording. Source WAV is 48 kHz stereo; the
# decoder auto-downmixes and decimates.
